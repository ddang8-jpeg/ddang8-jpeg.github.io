Proto-Object Modeling

Team member(s)

  • Khaled Aboumerhi

Neuromorphic Vision Sensors have recently garnered a lot of interest due to their low-power, highly efficient methods of capturing information without the redundancy and high power methods of typical cameras. Inspired by the mammalian retinal architecture, NVS cameras capture information only when a significant change in intensity cross a pixel threshold. The cameras send this data in the format of AER streams: address-event representation, where the format is as follows: [x-pixel, y-pixel, polarity of intensity change, time-stamp]

Using these silicon retina (DVS, ATIS, DAVIS), we can perhaps shed some light on visual saliency: features that grab attention, or, what feature stands out in a field of vision. Imagine yourself sitting at a bus stop reading a newspaper when a car accident happens. Would you continue to read your paper, or would you observe the accident? These are the sort of questions we would like to answer based on spike-based computing

Using incoming AER instances, we first built a saliancy algorithm that implements hand-crafted filters to track objects of desire. Then, using an online spatiotemporal learning method, we improved the algorithm to learn time-surfaces by convolving instances with different time kernels, and compare these time-surfaces with incoming data. Our current task is acquiring a saliency ground-truth data-set using the HTC Vive Pro Eye to compare the validity of our saliency algorithms.
